# JavaScript Basic Questions & Answers

## 1) null এবং undefined এর পার্থক্য কী?

**undefined** মানে ভেরিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু কোনো ভ্যালু দেওয়া হয়নি।  
**null** মানে ইচ্ছাকৃতভাবে ভেরিয়েবলকে খালি বা কোনো মান নেই সেট করা হয়েছে।

---

## 2) JavaScript এ map() ফাংশনের ব্যবহার কী? এটি forEach() থেকে কীভাবে আলাদা?

**map()** একটি নতুন array তৈরি করে এবং প্রতিটি উপাদানের উপর কাজ করে নতুন মান রিটার্ন করে।  

**forEach()** শুধু প্রতিটি উপাদানের উপর কাজ করে কিন্তু নতুন array রিটার্ন করে না।

---

## 3) == এবং === এর পার্থক্য কী?

**== (Loose Equality)** শুধু মান তুলনা করে, টাইপ কনভার্সন করে।  

**=== (Strict Equality)** মান এবং টাইপ দুটোই তুলনা করে, টাইপ কনভার্সন করে না।

---

## 4) API থেকে ডাটা আনার ক্ষেত্রে async/await এর গুরুত্ব কী?

**async/await** ব্যবহার করলে asynchronous কোড সহজ ও পড়তে সুবিধাজনক হয়।  
এটি promise হ্যান্ডেল করতে সাহায্য করে এবং কোডকে synchronous এর মতো লিখতে দেয়।

---

## 5) JavaScript এ Scope কী? (Global, Function, Block)

**Scope** হলো ভেরিয়েবলের কাজ করার ক্ষেত্র বা পরিসীমা।

- **Global Scope** → পুরো প্রোগ্রাম জুড়ে ব্যবহার করা যায়।  
- **Function Scope** → শুধুমাত্র নির্দিষ্ট ফাংশনের ভিতরে ব্যবহার করা যায়।  
- **Block Scope** → {} এর ভিতরে সীমাবদ্ধ (let, const দিয়ে তৈরি)।
